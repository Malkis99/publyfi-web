# Content & Interaction

This document outlines the primary user-facing components of the PublyFi platform, from the main navigation tabs to the secondary system tabs like the user profile.

## Main User Tabs (Primary Navigation)

1.  **Home (Feed)**
    *   Personalized recommendation feed based on games, interests, and user activity.
    *   Mixed content: streams, clips, `Reels`, posts, tournaments, news.
    *   Tag & filter system (game genres, `IRL`, movies/series, esports).
    *   Curated sections: Top of the Week, New Streamers, Best Quests.
    *   Social dynamics: likes, comments, shares, trending topics.

2.  **Streams**
    *   Live streams for games, `IRL`, movies/series (with moderation).
    *   Sub-categories: `MOBA`, `RPG`, `Shooters`, etc.
    *   Integrated donations & real-time quests inside streams.
    *   Streamer challenges: issue quests during live broadcasts.
    *   Interactive overlays: polls, quests, `NFT` drops.

3.  **Reels (Shorts)**
    *   TikTok/Instagram-style short video feed — fully gaming-themed (memes, challenges, moments).
    *   Recommendation algorithm based on user preferences.
    *   Fast clip editing directly from streams.
    *   Music, effects, filters, collaboration tools.
    *   Viral growth mechanics: hashtag challenges, duo-collabs.

4.  **Quests**
    *   Streamer-created tasks with rewards (`NFTs`, achievements, bonuses).
    *   Platform-generated events & seasonal challenges.
    *   Available only for `Pro` subscribers.
    *   Leaderboards for top quest performers.
    *   Core gamification loop of the platform.
    *   Tiered quest difficulty with escalating rewards.

5.  **Marketplace**
    *   `NFT` trading: cosmetics, sets, emotes, stickers.
    *   Item rarities: `Common` → `Rare` → `Epic` → `Legendary` → `Mythic`.
    *   Buy/Sell/Trade functionality.
    *   Auctions and limited-edition collectibles.
    *   Cross-seasonal drops (e.g., Cyberpunk set in Winter Season).
    *   Fusion mechanics for upgrading `NFTs` into higher rarities.

6.  **Events**
    *   Game tournaments.
    *   Seasonal festivals.
    *   Major game collaborations (exclusive `NFTs` for big releases).
    *   Online events for `Pro` users.
    *   `IRL` events: esports, festivals, community meetups.
    *   Live metaverse-style shows hosted by streamers and creators.

7.  **Community**
    *   Forums, discussions, guilds, clans.
    *   Group challenges.
    *   Mentorship system for new users.
    *   Collaborative quest creation by guilds.
    *   (Governance moved to `DAO` tab; see below).

8.  **$PUBL (Token Hub)**
    *   **Overview Metrics:** current price, circulating/total supply, market cap (if applicable).
    *   **Charts:** price/time, volume, on-platform velocity.
    *   **Burn Dashboard:** total burned from fees; live ticker.
    *   **Halving Countdown:** next halving date/time + historical halvings.
    *   **Transactions Feed:** recent transfers, mints/burns, quest payouts.
    *   **Actions:** buy/sell (`DEX`), bridge (multichain), deposit/withdraw, staking (if enabled).
    *   **Compliance:** token contract, audit links, risk disclosure.

9.  **DAO (Governance)**
    *   **Proposals:** list, details, discussions.
    *   **Voting:** on/off-chain, quorum, thresholds, timers.
    *   **Delegation:** assign voting power, view delegates, reputation metrics.
    *   **Treasury:** holdings, inflows/outflows, grants, reports.
    *   **Roadmap Governance:** feature prioritization, tokenomics, seasonal themes.
    *   **Archive:** past proposals and outcomes.

## Secondary / System Tabs (Outside Primary Nav)

### Profile (Footer Entry)
Accessed from bottom bar / user avatar (not in main nav).

**Tabs:**
*   **Overview:** identity snapshot, avatar, key stats.
*   **Streams:** archive.
*   **Highlights/Clips:** best moments.
*   **Reels:** short videos.
*   **Gallery:** screenshots, fan art, memes.
*   **Inventory:** `NFTs`, sets, cases.
*   **Stats:** detailed progression.

**Unique Features:**
*   **3D Living Avatar:** equipment display (sets unlock auras/animations).
*   **Social Reputation:** level, achievements, rare items.
*   **Stats Panel:** Followers, Following, Likes (on avatar), Completed quests, Progress bar, Active tier (`Free`/`Prime`/`Pro`).
*   **Customizable backgrounds & profile themes** via `NFTs`.
